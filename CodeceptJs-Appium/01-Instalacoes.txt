5 - Instalando o Appium
    Appium UIAutomator2
    5.1 - sudo npm install -g appium
    5.2 - instale o Appium desktop
        5.2.1 repositório: https://github.com/appium/appium-desktop/releases/tag/v1.22.3-4
        5.2.2 arquivo: https://github.com/appium/appium-desktop/releases/download/v1.22.3-4/Appium-Server-GUI-mac-1.22.3-4.dmg
            OBS: depois de tentar abrir uma vez o appium desktop será necessário entrar em 
            'preferências do sistema' > 'segurança e privacidade' (e autorizar o sistema a abrir o Appium desktop)
    5.3 - Instale o Appium UIAutomator2
        5.3.1 - Android Home > SDK Maneger > SDK tools --> instale o Android Auto API Simulators
        5.3.2 - Configure as varáveis de ambiente (item 8 desse manual de instalações)
        5.3.3 - Desligue o servidor Appium e execute a linha de comando: sudo appium driver install uiautomator2
        5.3.3 - Rode o appium-doctor que já é instalado com o UIAutomator2: appium driver doctor uiautomator2
    5.4 - Instalar as dependências do Appium UIAutomator2
        5.4.1 - sudo appium driver install xcuitest ---> iOS
        5.4.2 - sudo appium driver install uiautomator2 ---> Android
        5.4.3 - verificar se tudo está instalado: appium driver list

    Appium normal
    5.1 - sudo npm i --location=global appium
    5.2 - Appium-doctor
        Appium-doctor
        5.2.1 - sudo npm install -g appium-doctor (utilize o sudo para desbloquear com senha)
        5.2.2- rode o appium doctor no terminal com o comando: appium-doctor (para verificar o sistema)

6 - Instalar o Xcode
    Para o Macbook 2017 a versão do xcode disponível é a Xcode 14.2, para baixar-la é necessário:
    6.1 - entrar no site: https://developer.apple.com/br/support/xcode/
    6.2 - acessar a página: https://developer.apple.com/download/all/?q=xcode
        obs: será necessário entrar com a conta (e-mail) e a senha do iCloud (senha geral)
    6.3 - pesquise a baixe o Xcode 14.2

7 - Instalar Android Studio

8 - Configurar as variáveis de ambiente do Java e Android no mac
    8.1 - Exemplo de como ficou as variáveis de ambiente do java e Android (exemplo real)

    export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-22.jdk/Contents/Home
    export PATH=$JAVA_HOME/bin:$PATH

    export ANDROID_HOME=/Users/rodrigotozatti/Library/Android/sdk
    export PATH=$PATH=$ANDROID_HOME/tools:$ANDROID_HOME/emulator
    export PATH=$PATH=$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools
    export PATH=$PATH=$ANDROID_HOME/tools:$ANDROID_HOME/tools

    8.2 - saia do nano com os comandos: 
        control + x 
        y
        Enter 
    8.3 - salve as confogurações utilizando o comando: source ~/.zshrc
    8.4 - para garantir que a configuração está correta execute o comando: echo $JAVA_HOME
        o resultado tem que ser o caminho do java
